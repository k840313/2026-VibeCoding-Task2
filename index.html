關於我

我目前是一名專注於電商領域的前端工程師與技術教練。我擅長利用 HTML、CSS 與 Python 自動化工具來優化設計流程，並致力於打造兼具美感與轉化率的網頁介面，幫助客戶在競爭激烈的市場中脫穎而出。

在工作風格上，我極度重視與夥伴的溝通透明度。我喜歡將複雜的技術概念拆解成易懂的語言，並持續透過教學來打磨自己的專業技能，確保產出的每一個專案都能精準對齊商業目標並易於維護。

  • 高效自動化： 結合 Python 與前端技術，大幅提升重複性設計任務的產出效率。
  • 商業邏輯導向： 不只做美觀的設計，更從電商數據與行銷角度思考頁面佈局。
  • 清晰技術溝通： 具備跨團隊協作經驗，能將複雜需求轉化為可執行的技術方案。
